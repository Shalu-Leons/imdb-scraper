\# IMDb Movie Rating Scraper



The \*\*IMDb Movie Rating Scraper\*\* is a Python-based automation tool that dynamically scrapes movie

data from IMDb using Selenium and Chrome WebDriver.



\## Features

\- Scrapes IMDb Top 250 movies (title, year, rating, rank)

\- Saves data to CSV

\- Optional headless mode

\- Extendable to fetch more details (genre, cast, etc.)



\## Technologies

\- \*\*Python\*\*

\- \*\*Selenium\*\*, \*\*pandas\*\*, \*\*webdriver\_manager\*\*, \*\*time\*\*
