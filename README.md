#  IMDb Movie Rating Scraper

##  Project Description
The IMDb Movie Rating Scraper is a Python-based automation tool that dynamically scrapes movie data from IMDb’s **Top 250 Movies** list using Selenium and Chrome WebDriver.  
It retrieves details like **title, release year, IMDb rating**, and stores them in a structured CSV format.

##  Features
-  Dynamic web scraping using Selenium  
-  Extracts movie title, year, and rating  
-  Saves data to CSV  
-  Optional headless mode  
-  Expandable for detailed scraping (cast, genre, etc.)


##  Technologies Used
- **Python**
- **Selenium** – browser automation  
- **pandas** – data handling  
- **webdriver_manager** – auto ChromeDriver setup
